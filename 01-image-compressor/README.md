# img_compressor

Este un compresor de imagenes personalizado, surge de la necesidad de tener una herramienta de compresion personalizada para imagenes en la web, ademas permite la redimension y mejora o disminucion de la calidad.<br /><br />

Migracion a mcp para integrarlo para la optimizacion de imagenes en sitios estaticos Trabajo en progreso.
